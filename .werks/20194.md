# Restore presentation option for combined graph dashboard widgets

class      | fix

When adding a graph to a dashboard via _Metrics & graphs_ and choosing
_Graph (combined)_, there was no way to choose how the individual metrics are
combined into the graph.

This has been fixed. The _Additional graph options_ now offer a _Presentation_
dropdown again, letting you choose between _Lines_, _Stacked_, _Sum_, _Average_,
_Minimum_ and _Maximum_.

Existing dashboards are not affected: their stored presentation is kept and is
now editable in the user interface.
