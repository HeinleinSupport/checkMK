# (Windows) HW/SW Inventory: Translate product language code

key        | value
---------- | ---

version    | 2.6.0b1
class      | fix
edition    | community
component  | checks

Software packages previously displayed the product language code (eg. "1033")
in the language column. Now, the language code is translated to show the full
language name (eg. "English") in the language column.

Please note that this change also generates a HW/SW Inventory history entry.
