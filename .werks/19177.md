# aws_ec2_limits: Added Several AWS EC2 Resource Types

Several resource types were added into the aws_ec2_limits check.

These include:

* r6a
* r6i
* r6id
* r6idn
* r6in
* r8g
* r8gd

Before this change, the check would report "Unknown resource".

With this change applied, the limits will be reported correctly.
