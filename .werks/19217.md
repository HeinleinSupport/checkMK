# Windows Updates: Override and remove monitoring state, rename reboot notification option

---------- | ---

version    | 3.0.0b1
class      | feature
edition    | pro
component  | checks
level      | 1
compatible | yes

Previously there was no way to override or remove the monitoring state of
Windows Updates through the GUI. Additionally, the option *Levels for time
until forced reboot due to pending important updates* had a misleading name
that did not accurately describe its purpose.

Starting with this release, users can override and remove the monitoring state
of Windows Updates directly in the GUI. The option *Levels for time until forced
reboot due to pending important updates* has been renamed to
*Windows Update notification time* to better reflect its function. No action is
required — existing configurations continue to work unchanged.
