# Fix Redfish outlet monitoring for alphanumeric outlet IDs

key        | value
---------- | ---
date       | 2026-02-17T08:17:00+00:00
version    | 2.6.0b1
class      | fix
edition    | community
component  | checks
level      | 1
compatible | yes

Previously, the Redfish Outlets check failed to identify outlets from PDUs that use alphanumeric IDs like "L3_F2_30".
This format is common for three-phase PDUs such as Rittal devices.
The outlet services would show no data because the ID could not be matched.
This has been fixed.
